# Level 5 - if文
- Level 4では、円を下げることに成功したが、下がるだけでエリアから出て行ってしまった
- 描画エリア内に戻すにはどうすればいいだろう？ 
    - if文の出番

- if文とは
    - このような書き方のこと
        ```
        if (条件) {
            やりたい処理
        }
        ```
    - 条件に当てはまるときだけ、処理を行うことができる

- 実際に書いてみる
    - このようなif文を足して、動かしてみよう
        ```
        if (y > 400) {
            y = 0;
        }
        ```
    - コード全体
        ```p5.js hl_lines="10-12"
        let x = 200;
        let y = 0;

        function setup() {
            createCanvas(400, 400);
        }

        function draw() {
            y = y + 3;
            if (y > 400) {
                y = 0;
            }
            // background(220);
            fill(255, 0, 0)
            circle(x, y, random(0, 50))
        }
        ```
        - ちなみに、background() は有効化してもOK
        - 下の辺に触れたら、上から現れるようになったはず！

- どういう意味か？
    - 再掲
        ```
        if (条件) {
            やりたい処理
        }
        ```
    
        ```
        if (y > 400) {
            y = 0;
        }
        ``` 

    - 条件: `y > 400`、つまりyが400より大きくなったら
    - 処理: `y = 0`、つまりyを0にする
    - まとめると……
        **yが400より大きくなったら、yを0にする** という意味
    - 400という数字は描画エリアの縦の高さ
        - 気になったらこの数字を変えて遊んでみよう
