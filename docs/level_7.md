# Level 7 - for文

- 「配列」でcircleの描画を複数回行いやすくなったが、もし99個出したいなら99行書く必要があり、これは面倒
    - ここで「for文」の出番
    - プログラミングでは「これは面倒だな、もっと別のやり方ないのかな」と思った場面では、だいたい何か賢いやり方がある
- for文: 指定した回数や、条件を満たしている間、処理を繰り返すことができる
    - 「一定の規則で繰り返されるもの」はfor文で実装するのが良い

- for文の書き方

    ```p5.js
    for (初期化式; 条件式; 変化式) {
        繰り返しやりたい処理
    }
    ```

    - 初期化式・条件式・変化式
        - 初期化式: ループを開始するときに1回だけ実行される
        - 条件式: 処理を繰り返すかどうかを判定する
        - 変化式: 処理が1回終わるたびに実行され、変数の値を更新する

    - 例

        ```p5.js
        for (let i = 0; i < 3; i = i + 1) {
            circle(x[i], y, random(0, 50));
        }
        ```

        ![for文のイメージ](images/for.png)

    - 難しいので、書くことで慣れていこう

- for文を使って円を3つ出すプログラム

    ```p5.js hl_lines="15-17"
    let x = [100, 200, 300];
    let y = 0;

    function setup() {
        createCanvas(400, 400);
    }

    function draw() {
        y = y + 3;
        if (y > 400) {
            y = 0;
        }
        background(220);
        fill(255, 0, 0);
        for (let i = 0; i < 3; i++) {
            circle(x[i], y, 50);
        }
    }
    ```

- Level 6の「配列」では `circle()` は3行書いていたが、1行で済んだ！
    - Level 6

        ```p5.js
        circle(x[0], y, 50);
        circle(x[1], y, 50);
        circle(x[2], y, 50);
        ```

    - Level 7

        ```p5.js
        for (let i = 0; i < 3; i++) {
            circle(x[i], y, 50);
        }
        ```

    - こうすると変更がしやすいプログラムになる
