# Level 6 - 配列

- [Level 4](level_4.md) で変数について学んだが、たくさんのデータを作って使いたいとき、データそれぞれに変数名を割り当てるのは面倒
    - 「配列 (Array)」の出番
- 配列の書き方
    - `x = [100, 200, 300]`
    - 複数の値を1つの変数に入れられる
- 使い方 (配列内の**要素**の呼び出し方): `x[0]` や `x[1]`など
    - プログラミングでは1番目のことを `[0]` で表すことに注意

- 配列を使って円を3つ出すプログラム

    ```p5.js hl_lines="1 15-17"
    let x = [100, 200, 300];
    let y = 0;

    function setup() {
        createCanvas(400, 400);
    }

    function draw() {
        y = y + 3;
        if (y > 400) {
            y = 0;
        }
        background(220);
        fill(255, 0, 0);
        circle(x[0], y, 50);
        circle(x[1], y, 50);
        circle(x[2], y, 50);
    }
    ```

    - たとえば `x[0]` では `100` を取得しているので、`circle(100, y, 50)` と書くのと同じ
